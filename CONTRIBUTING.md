# Contributing to `auto-qa`

Thank you for considering contributing to `auto-qa`!

## Local Development

To set up your local development environment, follow these steps:

1. **Install Python, `just`, and `uv` (recommended via Homebrew):**
   ```sh
   brew install go just prek
   ```

2. **Initialize `just` in the project (if not already done):**
   ```sh
   just init
   ```

3. **Run tests:**
   ```sh
   just test
   ```

4. **Start development:**
   - Make your changes and use `just` commands as needed like `just test`, `just run`, or `just rund`
